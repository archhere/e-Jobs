
# e-Jobs

A marketplace app for bidders and posters


## Installation

Check your Node version and make sure it's `14.21.1`:

```
$ node -v
v21.7.1

$ npm -v
10.7.0


### How to run
Server side:
CD into /api and run the following commands.
npm i
npm run start 
npm run test

client side:
CD into /client and run the following commands.
npm i
npm run start
npm run test
